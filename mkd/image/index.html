画像の表示方法

画像は基本的に Markdown 記法で貼りつける。

外部サイトの画像ファイル

表示したい画像の URL をそのまま指定する。

例: ![ALT](https://bagof.crap.jp/img/vr/daisy1.jpg)

サイト内の画像ファイル

ソースの段階で docs/img/ にあるファイルは、ビルド後に site/img/ に入る。
これを Makrdown では img/ からの相対パスとして指定する。

例: ![ALT](img/daisy3.jpg)

拡大縮小は自力<img>タグで

Markdown 記法は画像サイズを指定できないため、サイズを変えたいときは自力で <img> タグを書く。 このとき サイト内の画像 URL は ../ から始まる ので注意する。

例: <img src="../img/daisy3.jpg" alt="ALT" style="width:80px; height:80px;">

<img width="80" ...> のように width 属性も使えるが、 height 属性は無視される（CSSでそう設定されている）。

凝った表示はすべて自力HTMLで

レイアウトに凝りたい場合などは自力で HTML を書く。 たとえば「4枚の画像を横に並べ、レスポンシブ対応で拡大縮小」の例。

<style>
.image-container {
  display: flex;
}
.image-container img {
  width: 25%; /* 4枚で100% */
}
</style>

<div class="image-container">
  <img src="../img/daisy1.jpg" alt="画像1">
  <img src="../img/daisy2.jpg" alt="画像2">
  <img src="../img/daisy3.jpg" alt="画像3">
  <img src="../img/daisy4.jpg" alt="画像4">
</div>
