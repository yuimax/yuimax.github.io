初期設定

開発環境の準備

本サイトの想定環境は下記の通り。 統合開発環境も AI エージェントも必要ない。 たいへんお手軽である。

  • Windows 11
  • テキストエディタ
  • コマンドプロンプト
  • Python 3
  • GitHub 無料アカウント

すでに GitHub のアカウントは取得済であることを前提とする。 無料アカウントでよい。

インストール方法

Pythonのインストール

Python は必須。
https://www.python.org/ から最新版を落としてインストールする。

MkDocs本体のインストール

Python を入れると pip コマンドが使えるようになる。 これで MkDocs 本体をインストールする。

pip install mkdocs

インストールに成功すると mkdocs コマンドが使えるようになる。 次のように "-V" オプションでバージョンが表示されればオーケーである。

mkdocs -V
↓
mkdocs, version 1.6.1 from ... (以下略)

多国語拡張のインストール

UI を日本語化したい場合は、i18n版 もインストールする。 ちなみに i18n とは i(18文字)n の意味で、internationalization の略である。

pip install "mkdocs[i18n]"

プラグインのインストール

MkDocs のプラグインもインストールする。 いま利用しているものを下記に示す。 プラグインを追加したら、その都度ここに書いていく。

mkdocs-macros-plugin: ファイルのインクルード

pip install mkdocs-macros-plugin
