■VRoid Daisy 1.0.0

  • 分類： VRoid Studio + Vroid2Pmx + PmxTailor による MMD モデル
  • 説明： MMD練習ノート/69
  • 配布： https://bowlroll.net/file/299134

●おもなファイル

  • daisy.pmx // MMDモデル（物理あり）
  • daisy.vroid // vroidモデル
  • daisy_nophysics.pmx // MMDモデル（物理なし）
  • 03Bot_根本.csv // 物理なしスカートの根本の頂点座標
  • 03Bot_裾.csv // 物理なしスカートの裾の頂点座標
  • 03Bot_物理.csv // 物理なしスカートの全体の頂点座標
  • [PmxTailorSetting] // PmxTailor用の設定ファイル
  • [tex] // テクスチャ画像

●PmxTailor で別の物理を入れる

  • 現在 daisy.pmx はスカートに [コットン] の設定を入れています。
  • PmxTailor で別の物理設定を試すこともできます。
  • 方法は作者ブログの 説明記事 をご覧ください。

●使用条件

  • 利用方法について特に制限はありません。
    R-18、R-18Gにも利用可能です。
    改変、流用、再配布なども自由です。
  • 本モデルを商業利用することはできません。
    これは作成にあたり使用した「Pmxエディタ」の規約です。
