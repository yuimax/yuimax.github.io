■VRoid Daisy 1.1

  • 分類： VRoid Studio + Vroid2Pmx + PmxTailor による MMD モデル
  • 説明： MMD練習ノート/108
  • 配布： https://bowlroll.net/file/299134

●おもなファイル

  • daisy.pmx // MMDモデル Vroid Daisy v1.1
  • daisy.fx // 中身は Elle/データP の AdultShaderS2.fx
  • [tex] // テクスチャ画像
  • daisy.vroid // vroid モデル
  • daisy_z.pmx // おまけ（ずんどう版）

●シェーダーファイルについて

  • Elle/データP 作成の AdultShaderS2.fx の設定を変えたもの
  • 再配布が許可されているので、添付させてもらいました

●おまけ（daisy_z.pmx）の使い方

  • 試験的にずんどうモーフを入れたバージョン
  • 上下の服を脱いで、モーフ [Bスーツ] を1にするとずんどう化します
  • 服を着たままだと破綻します
  • VRoid Studio の「ボディースーツ」を重ね着しているだけです

●使用条件

  • 本モデルの使用、改変、流用、再配布は自由です
  • 本モデルを商業利用することはできません
    これは使用ツール「Pmxエディタ」の規約です
  • 本モデルは R-18、R-18G の利用も自由です……が、
    使用ツール「Vroid2Pmx」「PmxTailor」の説明書に、
    「こうした用途は禁止ではないが配慮ください」との記述あり
